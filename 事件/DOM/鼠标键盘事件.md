```
鼠标点击事件click 点击事件唯一性保留后面的的
鼠标经过事件onmouseover
鼠标离开事件onmouseout
获取鼠标焦点onfocus 没有冒泡
失去鼠标焦点onblur 没有冒泡
鼠标移动触发onmousemove 没有冒泡
鼠标弹起触发onmouseup 没有冒泡
鼠标按下触发onmousedown

禁止鼠标选中selectstart
```

```
鼠标事件对象
e.clientX //可视区x轴距离
e.clientY //可视区y轴距离
e.pageX //相当于页面的x轴距离
e.pageY //相当于页面的y轴距离
e.screeX //相当于屏幕的x轴距离
e.screeY //相当于屏幕的y轴距离
```
