```
属性
location.href 获取或者设置整个url
localtion.host 返回主机
localtion.port 返回端口号
localtion.pathname 返回路径
localtion.search 返回传输的参数
localtion.hash 返回片段
```

```
方法
location.assign() 和href,可以跳转页面 重定向页面
location.replace() 替换当前页面，不记录历史，不能后退页面
localtion.reload() 重新加载页面相当于刷新f5 参数true强制刷新
```

```
navigator.userAgent 浏览器头对象
```

```
history
back() 后退功能
forword 前进功能
go(参数) 前进后退功能 参数如果是1前进1个页面 -1后退一个页面
```
