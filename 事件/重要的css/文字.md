```
font-style : 取值
作用:规定文字样式
取值:
normal ： 正常的， 默认就是正常的
italic ： 倾斜的

font-weight : 取值
作用: 规定文字粗细
取值
bold： 加粗
bolder： 比加粗还要粗
lighter： 细线， 默认就是细线

font-size : 取值
作用: 规定文字大小
取值
格式：font-size: 30px;
单位：px（像素 pixel）
注意点： 通过font-size设置大小一定要带单位， 也就是一定要写px

font-family : 取值
作用: 规定文字字体
取值
格式：font-family:“楷体”;
注意点：
1.如果取值是中文， 需要用双引号或者单引号括起来
2.设置的字体必须是用户电脑里面已经安装的字体


```
